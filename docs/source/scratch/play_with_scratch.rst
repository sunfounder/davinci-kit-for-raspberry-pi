Spielen mit Scratch (nicht für Pi 5)
==========================================

Scratch ist eine blockbasierte visuelle Programmiersprache und Website, 
die sich in erster Linie an Kinder von 8 bis 16 Jahren als pädagogisches Werkzeug zum Programmieren richtet. 
Benutzer der Website können Projekte im Web über eine blockartige Oberfläche erstellen. 
Der Service wird vom MIT Media Lab entwickelt, 
wurde in mehr als 70 Sprachen übersetzt und wird in den meisten Teilen der Welt verwendet.

Hier lernen Sie, Scratch 3 im Raspberry Pi zu verwenden und mit Scratch 3 auf Raspberry Pi GPIO zuzugreifen, 
was mit Online Scratch nicht möglich ist.
